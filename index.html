Shop Jacket - Minimalist Product Page

This project is a modern, responsive e-commerce product page built using HTML and CSS as a practice exercise. The design follows clean and minimalist UI/UX principles, ensuring a sleek and user-friendly experience.

The page showcases a product listing with a high-quality image, engaging typography, and interactive elements like an "Add to Cart" button and expandable sections for product details and shipping information. The layout is fully responsive, adapting seamlessly to different screen sizes.

The project focuses on modern CSS techniques like Flexbox, Grid, and custom styling to achieve a polished and professional look. Every design choice, from spacing to typography, was made to enhance readability and user engagement.

This project was completed in just two days and is now live on GitHub Pages, demonstrating my ability to quickly build and deploy visually appealing web interfaces.
